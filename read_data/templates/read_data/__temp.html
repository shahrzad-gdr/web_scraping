Web Scraping Project

In this project, we learned how to extract data from the Top 250 Movies list on the IMDB website, it into a database, and display it to the user.

You will also learn how to use the Beautiful Soup library, manipulate user blocks in HTML files, retrieve data from a database using queries, and incorporate Bootstrap into your project.
